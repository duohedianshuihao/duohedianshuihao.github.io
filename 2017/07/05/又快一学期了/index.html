又快一学期了

发表于 2017-07-05 | 分类于 假装很忙 |

距离上一次说要更新，已经过了差不多一个月了吧。一是觉得自己没时间，每次写东西就写的太长了，二是觉得没什么好写的。就算心里有挺多的东西想说，一打开电脑也就不知道说什么了。哦那说白了就是又懒又笨。好吧，可能就是这样。

转眼这一学期也就剩下一个月了。学期结束，就可以回家带一个月了。其实我不怎么想家，但是还是很想回去一趟，还是充满期待，虽然回去了也还要好好准备冬季找实习，但是能回家，我还是感觉很放松。

听说春季学期很水

春季学期还是比较好上的，课比较水，总之水一水就过去了。 —— 忘了谁说的了

我tm反手就是一巴掌。看着我的眼睛，不水！这学期选了一门讲机器学习的课，还作死选了一门cs的分布式。机器学习那么还好，但是要写的东西太多了，project还要提出新的观点，要做到能发论文的水平。当时我就乐了，小老头你特么自己瞧不起自己？我一个屁也不懂的，看见大串的数学公式就脑袋疼，然后你给我一个月时间，我就能想出一个新的想法，然后还给你实现了，再写个论文出来？我要是有那道行，我还真就不上你的课了…好在这个课不考试，就是两个论文实在让人头疼，而且还要看论文。

然后就得说说分布式了，到现在为止，学的东西都是协议，这些东西都还好，看看也能看懂。但是这个课的project就难为我了，看个clojure实现的测试框架，然后每周找小胖老师报告进度。我好像想起了被大创支配的恐惧……不过还好小胖人比较好，每次我都是抱着练口语的心态去的……

所以这学期，我开始抱着比较水的心态在上，没事刷刷题，健健身，然后看看其他东西。突然就到了6月多，发现不是那么回事……然后就突然感觉好多事情忙不过来了…当然这里面自己吓自己的成分比较多，主要是未知的东西不知道自己要用多久去学习，然后计划不好的事情就会让人焦虑。

说点高兴的

这学期比较高兴的事算是转专业成功了吧，终于如愿去了ece，不用为了9月份学三门syde的课操心了。我是一点也不想学什么图像处理，如果可以，多选点软件相关的课。然后惊喜的发现上学期653的老师又教650了，虽然他给分低点，要求严格点，但是起码他讲话比较清楚啊。这学期我听法语口音的英语，咖喱味的英语，真是听得够够的了。

然后这学期健身比较多了，毕竟买了蛋白粉了，花了钱了就是不一样。就像你想学摄影，就必须先买个好点的单反，不然你一定坚持不下去: ) 我惊喜的发现我现在已经150斤左右了，不知不觉来了以后瘦了10斤。昨天又惊喜的发现，引体向上，尽管不标准，可以做10个了。还是很开心的。

其实最近还是比较放纵的…

给大家推荐一个几分钟解说电影的up主，b站搜索木鱼水心，我感觉说的很好，我很喜欢。声线比较好听，内容也偏正经。

然后在他说了legal high之后，我无法自拔义无反顾的看完了全部两季+两个special。我真的是大力安利这个剧，当然如果你跟我一样，没什么定力，看一个剧绝不接受看一半先不看了。那我觉得你还是等有时间了再看，毕竟大概率你会喜欢上。

题材轻松搞笑，有时候又能给你灌点鸡汤，一点不一样的观点。我先给端一碗鸡汤上来给你们尝尝咸淡

哦对了，再看第二部的时候，心里莫名的对自以为是的圣母婊产生了由衷的鄙视，真的是由衷的鄙视。套用“罗老师”的一句话就是纯臭傻逼！在我就总能碰到这样的能人，以为自己上帝视角，看懂了一切。还好不在我身边…还好还好。

以上

好了，今天早晨起来，突然收到邮件，小胖一岁大的宝贝闺女发烧了，没办法来上课了，他感到十分抱歉。(大丈夫森赛！家庭要紧！)所以莫名觉得今天比较轻松。剩下的一个月应该抓紧一点，好好学习，好好健身。暑假回家，好好刷题，好好相处。就酱。

# 快回家了 # 我好懒散
第一学期就结束了
