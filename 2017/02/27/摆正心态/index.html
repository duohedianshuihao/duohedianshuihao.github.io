摆正心态

发表于 2017-02-27 | 分类于 拧巴 |

好了今天来填坑，想说的话其实就是一句话，来这里两个多月了，看不到自己的进步，心里有点失落。可能是自己心急了，也可能是自己对这里的新生活，对一个新的自己报的期望太高了。有时候会和自己说，不着急，慢慢来，一点点进步就可以。但是我感觉我不想再慢了。我想快一点。想快点让自己变得好一点，变得对自己硬一点。或者说让自己更有条理一点。

想学的东西还是没有看

真的给自己挖的坑太多了，但是自己真正填的坑又太少了。说实话我可不想变成一个什么都听说过，但是什么也不会用的人。Talking is cheap. 我觉得自己应该多学一些东西，有一项技能能够让自己找到工作。也算对得起自己了。

就拿写一个自己的博客来说吧，在家的时候就开始看，每天看一点点，自己也是再看了，但是又没什么实质性的进展，好像在磨洋工。然后来了这边之后就没再看过。开始的时候总说是刚来需要时间适应。现在已经两个多月了，要说适应应该也适应的差不多了。是不是可以考虑开始看看了。

爬虫真的会写了吗？pyspider波波给你安利以后你会了吗？找实习要不要去爬一下各种实习信息看一看？

这学期就两门课，数据库和软件测试。学的东西能不能用到你要写的辣鸡博客里来？SQL学了学，NoSQL是不是也看看？

写到这我发现我又给自己立了一堆的flag……

上课的感受

上课的东西还是远远不够的。我不是想鼓吹什么上课无用论，读书无用论，万般皆下品，唯有代码高这种比较无聊的言论。但是感觉课下的左右并没有带给我什么代码上的提高。然后就是最近还有了一个怪现象，就是来图书馆开始学习的时候（如果我每天不看youtube乱七八糟的东西，学习的时间多到我自己都害怕），我不想看书，不想看课件。与其看这些，我倒觉得不如去刷刷题。但是653期中考试的时候根本没写完，心里还是觉得有点失落，失落自己跟本科一样，没有认真对待考试，考试的时候还是能发现很多新知识……所以这就是我说的有点拧巴的心态。但你要让我说的话，我还是说不那么看重成绩。

然后就是老师还是很厉害的，Arie给我这种感觉。讲课清楚，耐心。技术不用说，真的屌。

多做自己觉得有意义的事

恩，就是要这样。摆好心态。
