迟到的菜谱

发表于 2016-02-20 | 分类于 个人菜谱 |

好吧，其实距离当时说要展示一下假期学习做菜的成果已经有过了大概半个月了，现在能拿得出手的也就还是那些，新学的大概就只有拔丝红薯和红烧鸡翅了。其实这两个菜并不实用……我都不知道国外吃不吃的到红薯，鸡翅更不用说了，有肯定是有，就是肯定贵的要死……现在就简单记录一下这几个菜的做法吧。

拔丝红薯

原料

红薯一块（最好能提前在阳台上晒一晒，那样比较甜，烤着吃会更好吃）、冰糖、油

制作步骤

  1. 红薯洗净削皮，切成大小适中的块（大小适中详见下面图片）
  2. 烧水，将切好的红薯放到锅里煮5分钟左右，煮到红薯稍微有点软了，捞出来，放到盘子里
  3. 平底锅(其它锅应该也可以)中放入少量油，油铺满整个锅底即可。小火，将红薯一块一块放入锅中，来回翻滚至表面坚硬即可。注意不要把红薯弄碎，要一个一个翻滚
  4. 炒勺放入一定量的油(没过底部多一点)，放入一把冰糖，大火，等到冰糖逐渐变成红色后，调成小火。将煎好的红薯放入油锅中，来回翻炒至所有的糖全部粘到红薯上
  5. 出锅装盘

红烧鸡翅

原料

鸡翅若干、葱一小段、姜两片、大料少量、花椒少量、料酒、酱油、盐、冰糖、油

制作步骤

  1. 鸡翅放入锅中，煮五分钟左右，捞出
  2. 炒勺中放入少量的油，并加入冰糖一小把就好，不用太多。大火翻炒至糖逐渐变红，调至小火，放入鸡翅。(注意放之前要把鸡翅的水倒一倒，不然放进去的时候油会溅出来)翻炒鸡翅至颜色稍稍变红
  3. 向锅中加入清水，至水没过鸡翅，调大火，放入切好的葱姜，放一粒大料，几粒花椒，一勺料酒，一勺酱油，一平勺盐。搅匀后，盖上盖子，焖五分钟左右（至汤汁冒泡）
  4. 小火，继续熬至锅里的汤汁没有了
  5. 出锅装盘

西红柿炒鸡蛋……

原料

西红柿两个、鸡蛋四个、葱一小段、姜两片、花椒少量、糖、油

制作步骤

  1. 将鸡蛋打入碗中，搅匀
  2. 炒勺中放入少量油(铺平底部即可)，放入花椒，油开后放入鸡蛋，翻炒几下之后放入盘中
  3. 炒勺放入少量油，放入西红柿，翻炒几下后放入鸡蛋。放入葱姜和少量糖
  4. 盖上锅盖焖一小下，到西红柿有汤汁，翻炒几次，注意将鸡蛋弄碎
  5. 出锅装盘
