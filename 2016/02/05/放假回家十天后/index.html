放假回家十天后

发表于 2016-02-05 | 分类于 文字 |

转眼回家已经十天了，放假的时间总是过得太快……但是这个假期过的还是挺舒服的，时间很多，也比较充实。尤其是开头的几天。有一说一，其实我真的还是喜欢有事情做的时候，喜欢那种充实的感觉。就像白天跟包子出来说的那样，衡中的那段时间，有奔头，能使上劲，感觉踏实。刚回来的那两天，换了一个环境，能感觉出来自己跟在学校不一样了，高兴。不管是学习了，还是做其他事情了，都感觉充实，不是在浪费时间。虽说现在有点松懈，但咱现在不是发现了吗，发现了就是好事，说明就有改的可能。

假期目标

课程设计

这个是假期最重要的事情了，现在完全还在准备阶段。learning python the hard way 才刚刚看了一半，而且感觉看完这个也没太大用……只能算是熟悉一下python，后面还有还多要看的东西。而且现在论文还没有翻译，相关的知识也还没有学。后面还有多少麻烦事还不知道，所以假期一定要努力多做一点，争取把核心的东西做出来。

那个可视化编程

现在都记得大二数据结构的时候波波给我看的那个可视化编程的那个视频。一直希望自己也能实现一遍。但是一拖再拖到了现在。这个假期一定要把它完成一下。对大神来说肯定不算什么，但是对我这种种初学者来说，还是挺有难度的。

学会使用简单的爬虫框架

Instagram上面有很多喜欢的图片，当时想保存下来，选择了ifttt+dropbox的方法。这个方法感觉比第三方的软件比格高。当时还挺满意的。后来觉得其实可以写一个爬虫都抓下来。难的写不了，就先来个简单的，把一个人的所有图片都抓下来总要容易些吧。会面慢慢在改进。

做饭

老妈终于答应让我多上手做做饭了，但是过去了十天了，也就做了个拔丝红薯。包饺子擀皮这种自己早都会了……等多做几道菜了，再来一篇专门炫耀自己做的菜的。不如就定成三个吧，学会三个菜就发一篇食谱。将来出去不会做了还嫩从这看。不过我很好奇，出去上学真的有时间自己做饭吗？只是好奇，我个人还是觉得做饭是很有用的。我觉得这是独立必须具备的技能，所以我很看重这方面。还有就是小黑说的那句话，我觉得很有道理。“结婚之前，不能饿着”

练习口语

其实本来是打算回家就在网上找个歪果仁当口语老师，好好练习一下口语。毕竟出去之后，比起听不懂，张不开嘴的问题可能更加严重。而且将来找工作的时候，交流很重要。但是十天过去了……别说练了，连找都没找好……年前可能够呛了，但是年后最好还是能每天抽出半个小时跟外国人交流一下。争取能上半个月的课吧，恩。

熟悉双拼

这个我现在就在做啊……这篇博客就是用双拼在写，微软双拼。这个过程真的是超级痛苦啊！！！一个小时过去了，一篇博客还没有写完！！！不过到了现在，基本上已经能够盲打了。这个感觉就像是打断手筋，重新修炼一样。希望双拼能有所裨益吧。如果不能，那就让我装逼吧……

锻炼身体

本来这个假期打算跟灏哥一起健身房走起的，但是灏哥腰好像出了点小问题。这个事已经彻底泡汤了。但是我觉得自己还是应该多为自己那个被肥肉绑架了的腹肌多考虑考虑。暂时先定个小计划吧。跑步、仰卧起坐、深蹲、俯卧撑，每天坚持！是不是有点儿耳熟？这是埼玉老师的诀窍吗！是啊。不过我坚持不下来啦，所以我当不了英雄啊==#

早睡早起

主要是早睡。不多说，12点之前上床。早点睡，早点起，挺好的。

以上所有，坚持一个假期

吐槽假期

老妈托人找了一个医术高明的中医给我看脸上的痘痘，整个假期都要吃中药。中药真的超级苦！！！没有吃过的我建议你们一定要尝一尝，会让你们对黄莲有更深的理解。讲真。苦也就算了，两天也就适应那个味道了，现在已经做到面不改色心不跳了，喝完在马上吃颗糖，问题不大。但是整个假期，牛羊肉不能吃，海鲜不能吃，辣椒不能吃(主要是吃饺子不能吃蒜)，就不能喝，不能喝凉饮料，不能吃凉水果。总之不能吃的东西太多了，多到我认为自己白回家了！就希望最后中药没白吃……

本命年喽

虽然我才23，但是明年就是猴年了，本命年。红裤衩已经就位！希望这一年能够顺利一些，能够有学上，能够顺利毕业。当然更希望家人能够身体健康。也希望自己能够更加了解自己。每个人都感觉自己了解自己，但是有时候可能自己并没有想象中那么了解自己，至少没有那么明白。我就是。比如最近晚上没有学习，跟家里一起看电视剧，八台的那个你是我的眼，沈腾演的那个。不知道怎么着就想到枪虾和虾虎鱼了。为什么自己会对白夜行那么情有独钟，大概是因为两个人之间的那份关系，那份彼此都相信这是世界上最可靠的东西的信念。无论如何两个人都从来没有动摇过。我觉得这样的感情才是命中注定的。哎，感觉自己还是没办法表达出来那种感觉。怎么说呢，就像是完全放心的把背后交给对方，或者说是做好了两个人一起面对将来要发生的所有困难却不会逃走的准备，大概就是这样吧。恩，这就是我现在的想法，希望自己将来也能找到像这样的感情。还有就是发现自己一看那种情感类型的，就很容易被带入进去，结果不好的虐心，结果好的又很快能肥肠冷静(类似贤者时间？)地分析出，这种过于完美的事情跟自己没太大关系。我的结论就是，没有什么感情戏的类似一拳超人那样的最适合我了！哈哈哈！

哦还有就是中间插播的那个李易峰的广告恶心死了……每次广告一出来我的心路历程是这样的。遥控呢！！！然后在沙发上痛苦的打滚，承受全部伤害以后，终于找到了……目光呆滞的骂他一句SB

最后祝所有人，猴年大吉，身体健康，万事如意！
